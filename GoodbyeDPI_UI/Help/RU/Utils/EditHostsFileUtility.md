# Изменение списка HOSTS на вашем ПК

Начиная с версии 0.5.0.0 в приложение встроена утилита, позволяющая заменить файл HOSTS на вашем ПК. 

В процессе использования утилиты может мерцать окно консоли.

### Зачем это нужно?

Изменение файла HOSTS позволяет решить следующие проблемы: 
1. Исправить подключение к серверам Discord, расположенным в Финляндии.
2. Вернуть доступ к серверам GitHub, которые необходимы для получения обновлений приложения и корректной работы Магазина.

Подробную информацию о том, как работает файл HOSTS и его восстановлении в Windows можно найти по адресу: [форум поддержки Майкрософт](https://support.microsoft.com/ru-ru/topic/c2a43f9d-e176-c6f3-e4ef-3500277a6dae).

### Как добавить адреса в файл?

*Примечание:* Если пункт меню [Добавить адреса в файл hosts] недоступен, то попробуйте сначала восстановить файл, а затем повторить попытку.

1. Откройте [Утилита работы с файлом hosts]. Сделать это можно на странице [Утилиты] в главном окне приложения.
2. Выберите [Добавить адреса в файл hosts].
3. Согласитесь с предоставлением прав администратора процессу `EditHostFile.exe`. Это обязательно, иначе изменить файл не получится.
4. Убедитесь, что статус замены в верхней части окна утилиты отображается как "✅ Файл hosts был изменен".
5. Мои поздравления!

### Как удалить адреса из файла?

*Примечание:* Если пункт меню [Убрать адреса из файла hosts] недоступен, то изменения в файл hosts не производились.

1. Откройте [Утилита работы с файлом hosts]. Сделать это можно на странице [Утилиты] в главном окне приложения.
2. Выберите [Убрать адреса из файла hosts].
3. Согласитесь с предоставлением прав администратора процессу `EditHostFile.exe`. Это обязательно, иначе изменить файл не получится.
4. Убедитесь, что статус замены в верхней части окна утилиты отображается как "❎ Файл hosts не изменен".
5. Вы неотразимы!

### Как восстановить файл?

*Примечание:* Если пункт меню [Восстановить оригинальный файл hosts из резервной копии] недоступен, то изменения в файл hosts не производились.

1. Откройте [Утилита работы с файлом hosts]. Сделать это можно на странице [Утилиты] в главном окне приложения.
2. Выберите [Восстановить оригинальный файл hosts из резервной копии].
3. Согласитесь с предоставлением прав администратора процессу `EditHostFile.exe`. Это обязательно, иначе восстановить файл не получится.
4. Убедитесь, что статус замены в верхней части окна утилиты отображается как "❎ Файл hosts не изменен". И кнопка восстановления стала недоступной для взаимодействия.
5. Превосходно!


## Возможные ошибки 

Антивирусы могут пометить файл `EditHostFile.exe` как вредоносный и удалить его. В случае удаления файла приложение будет выдавать ошибку вида `APPLICATION_DAMAGED_NEED_REPAIR` при попытке изменить файл hosts. 
В таком случае можно скопировать файл `EditHostFile.exe` из актуального Portable-релиза в рабочую директорию приложения рядом с `CDPIUI.exe`. Для MSI-версии достаточно выполнить восстановление. 

Для ручной отладки можно запустить `EditHostFile.exe` через терминал с правами администратора и отследить консольный вывод. 

| Флаг запуска утилиты | Описание |
| -------------------- | -------- |
| /add | Добавить адреса в файл hosts |
| /remove | Удалить адреса из файла hosts |
| /recover | Восстановить адреса из файла hosts.bak |

## Для разработчиков

В утилиту встроен файл hosts из репозитория [zapret-discord-youtube](https://raw.githubusercontent.com/Flowseal/zapret-discord-youtube/refs/heads/main/.service/hosts). Любые его изменения влекут за собой персборку утилиты (из соображений безопасности).

Добавляемые адреса помечаются тегом `[CDOM-B]` в начале и `[CDOM-E]` в конце добавляемого списка для упрощения их последующего удаления и редактирования.
