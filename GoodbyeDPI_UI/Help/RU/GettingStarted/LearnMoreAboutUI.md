# Что такое CDPI UI?

CDPI UI — это новая версия популярного графического интерфейса, ранее известного как GoodbyeDPI-UI. В новой версии была произведена работа над множеством ошибок, а также улучшены стабильность и скорость работы приложения. 

Основная цель CDPI UI — сделать доступ к таким утилитам, как Zapret и ByeDPI, проще и удобнее для современного пользователя. 

## Простой, но мощный

CDPI UI никак не ограничивает возможности компонентов. Если вы умеете самостоятельно настраивать такие компоненты, как Zapret или ByeDPI — дерзайте! 
Помощник создания пресетов позволяет создавать новые пресеты с удовольствием. Приятная для глаз подсветка флагов, возможность создания переменных для 
легкой интеграции с UI созданы специально для продвинутых пользователей. 

## Автономный

Механизм автоподбора настроек запуска стал ещё стабильнее. Новый интерфейс позволит собрать из нескольких рабочих стратегий для 
нескольких списков сайтов одну общую, без необходимости искать новую стратегию в Интернете.
Офлайн-справка позволит вам получить поддержку даже без подключения к Интернету.

## Такой, какой вам нужно

Магазин компонентов позволяет выбирать только те компоненты и наборы пресетов, которые вам нужны. Ничего лишнего.

## Бесплатный навсегда

Никаких подписок — вся функциональность всегда доступна без ограничений. Техническая поддержка также полностью бесплатна. 

Любая денежная поддержка разработчика — только по вашему личному желанию, без навязывающих баннеров.

## Открытый

Вы можете в любой момент изменить что-либо в коде приложения или удостовериться в его безопасности в официальном репозитории GitHib.

## Быстрый 

Приложение использует менее 15 Кб оперативной памяти при работе в фоне.
