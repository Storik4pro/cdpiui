# Добавление своих URL-адресов для обхода

Чтобы изменить список сайтов/IP-адресов, для которых будет выполняться обход, необходимо перейти к настройкам пресета используемого компонента. 

Если используемый пресет использует списки сайтов, то вы увидите виджет [Используемые списки сайтов]. В этом виджете перечислены все сайты, для которых будет выполняться обход. 
Для их изменения достаточно нажать на кнопку [Изменить] рядом с названием списка сайтов/IP-адресов. Для редактирования будет открыто настроенное вами средство редактирования .txt-файлов. 

Вы так же можете просмотреть, какие именно аргументы применяются к тому или иному списку сайтов, нажав на кнопку [Показать применяемые параметры] рядом со списком сайтов/IP-адресов.

### Обратите внимание
Списки сайтов/IP-адресов не являются уникальными для каждого пресета. Изменяя список сайтов/IP-адресов, он изменится для всех пресетов набора 
(исключение: набор "Локальные данные текущего пользователя" StoreId:`LocalUserStorage`).

### Для продвинутых пользователей
Узнать рабочие директории наборов пресетов можно через Магазин. Перейдите на страницу [Библиотека], нажмите кнопку [Показать больше действий] у желаемого элемента и выберите 
[Показать локальные файлы] в выпадающем списке.
