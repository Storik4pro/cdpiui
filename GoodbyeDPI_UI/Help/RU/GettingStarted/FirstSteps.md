# Первые шаги

Это руководство позволит вам изучить основы использования CDPI UI и настроить его под себя.

- **Скачайте первый компонент из Магазина.** Перейдите в Магазин компонентов, чтобы скачать нужный вам компонент. На момент написания этого руководства готовые наборы пресетов существуют только для Zapret. 
ByeDPI требует подбора пресета через GoodCheck (найти GoodCheck можно в разделе "Надстройки" Магазина).
	
- **Подготовьте скачанный компонент к работе.** Перейдите на страницу настройки установленного компонента и выберите предпочитаемый пресет (компонент автоматически перезапустится после изменения пресета, если
ранее он был запущен). Также не забывайте, что некоторые приложения и службы требуется перезапустить для получения достоверной информации о их доступности.

- **Настройте автозапуск компонента.** Для настройки автозапуска необходимо произвести несколько простых действий. Для начала включите автозапуск графического интерфейса, используя переключатель 
[Открывать приложение при запуске ПК] на странице [Параметры] в главном окне. Затем перейдите к настройкам установленного компонента и поставьте галочку [Запускать автоматически] в положение ВКЛ.

Примите мои поздравления! Настройка приложения была успешно завершена.

Если в будущем некоторые сервисы станут недоступны, то:

- **Проверьте наличие обновлений.** Перейдите на страницу [Загрузки] в Магазине и проверьте наличие элементов, которые следует обновить. Обновите их, если это потребуется.

- **Подберите пресет.** Снова пройдитесь по списку пресетов — может, один из них заработает.

- **Воспользуйтесь подбором через GoodCheck.** Подберите рабочий пресет через GoodCheck — перейдите на страницу [Утилиты] и запустите подбор, нажав на [Автоматическая утилита создания пресета]. 
Для подбора необходимо иметь установленную надстройку "GoodCheck".
