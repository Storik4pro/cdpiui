# Что такое автоподбор?

Автоподбор позволяет подобрать подходящие параметры запуска компонентов в автоматическом режиме без вашего участия. 

Существует несколько видов автоподбора:
- **Создание нового пресета с использованием GoodCheck.** Позволяет создать новый индивидуальный пресет, который позволит получить доступ к недоступным ресурсам.
- **Подбор лучшего пресета.** Позволяет подобрать лучший пресет из уже загруженных/созданных.

### Какой автоподбор стоит выбрать?

Создать новый пресет с использованием GoodCheck следует если:

- Вы уверенный пользователь ПК.
- Вы знаете, как работают блокировки и компоненты, которые их обходят хотя бы на базовом уровне. *Получить информацию по этому поводу можно на страницах компонентов.*
- Ни один из доступных в Магазине пресетов не позволяет получить доступ к требуемым сайтам.

Подобрать пресет из уже загруженных следует если:

- Вы неуверенный пользователь ПК.
- Вам неудобно перебирать пресеты вручную.

### Как мне начать автоподбор?

Чтобы узнать дополнительную информацию по этому вопросу посетите связанные страницы справки:

- Создание нового пресета с использованием GoodCheck
- Подбор лучшего пресета
