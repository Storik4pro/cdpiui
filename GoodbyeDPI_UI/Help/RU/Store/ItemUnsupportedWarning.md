# "Этот элемент на данный момент не поддерживается"

Обычно эту ошибку можно получить при попытке загрузить неподдерживаемый элемент из Магазина.

### Возможные варианты исправления 

- Обновите приложение до последней версии.
- Измените средство контроля версий Магазина на странице [Магазин/Параметры]. Это действие заставит приложение загрузить базу данных из
    другого источника, где может быть поддерживаемая версия этого элемента.

### Если ничего не помогает 

- Возможно этот элемент на данный момент не поддерживается приложением и его поддержка будет реализована в будущем. 
- Возможно произошла ошибка. Сообщите о ней в службу технической поддержки приложения, приложив необходимые журналы
    и подробно описав проблему (это важно!). Найти адрес технической поддержки можно на странице [Сведения] главного
    окна приложения.
