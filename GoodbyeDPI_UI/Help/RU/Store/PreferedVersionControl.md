# Средство контроля версий

Начиная с версии 0.3.0.0 Магазин получил возможность изменять предпочитаемый сервер для получения обновлений элементов и приложения.

### Для чего это используется?

Приложение постоянно связывается с сервером для получения следующих сведений:

- Наличие обновлений приложения.
- Наличие обновлений отдельных элементов, скачанных из Магазина.
- Последняя версия базы данных Магазина.

**[!] Обратите внимание.** Вы можете полностью отключить проверку наличия обновлений. Для этого перейдите в [Параметры] и выключите соответствующие флажки в группе [Уведомления].

Так же Магазин использует сервер для загрузки новых элементов на устройство.

### На что влияет изменение предпочитаемого сервера?

Изменение предпочитаемого сервера заставляет приложение проверять и загружать обновления с другого сервера. Однако некоторые элементы все ещё могут располагаться на внешних серверах вне зависимости от настроек приложения.

### Стоит ли мне изменить предпочитаемый сервер?

Вам стоит изменить предпочитаемый сервер, если:

- Скорость скачивания с текущего сервера крайне низкая.
- Приложение не может получить доступ к текущему серверу. Например: обновление базы данных Магазина не удалось с ошибкой `SSL`; проверка наличия обновлений приложения не удалась с ошибкой `UNEXPECTED_STATUS_CODE` и т.д.
- При попытке скачать элемент постоянно возникают ошибки связанные с сетью (`HTTP_REQUEST_EXCEPTION` и др.) **НЕ ПУТАТЬ** связанные с операциями записи (`IO_GENERIC` и др.) 

Если вы не испытываете проблем ни с Магазином ни с обновлением приложения, то изменять предпочитаемый сервер не стоит. *"Работает — не трогай!"*

### Как мне изменить предпочитаемый сервер?

Чтобы изменить предпочитаемый сервер выполните несколько простых действий: 

- Откройте Магазин компонентов (сделать это можно через страницу [Утилиты] в главном окне приложения).
- Перейдите на страницу [Параметры] Магазина.
- Переключите параметр \[Средство контроля версий\] на желаемый вариант.
- Перезапустите приложение: изменения вступят в силу только после перезапуска.

### Какие существуют средства контроля версий Магазина?

На данный момент Магазин имеет хранилище на открытых серверах GitHub и GitLab. В будущем количество доступных серверов может изменяться. 

- **GitHub** Репозиторий по умолчанию. Ссылается на оригинальные репозитории, расположенные на GitHub. Может быть недоступен для некоторых пользователей.
- **GitLab** Резервный репозиторий. Ссылается на резервные копии оригинальных репозиториев на GitLab. Резервные копии осуществляются разработчиком приложения и не имеют прямого отношения к авторам элементов.

Найти актуальную информацию о доступных серверах и их адресах можно в официальном Telegram-канале разработчика приложения.
