# Добро пожаловать в Магазин Компонентов

### Что такое Магазин?

Магазин — это удобный интерфейс для быстрого доступа к официальным репозиториям компонентов, надстроек и наборов пресетов.

### Это платно?

Нет, Магазин не взимает плату за доступ к бесплатному ПО, расположенному на GitHub/GitLab.

### Есть ли возможность автообновления установленных элементов?

Нет, Магазин руководствуется правилом *"Работает — не трогай!"*. Никакие элементы не обновляются автоматически до того момента, пока вы сами не решите их обновить.

Однако, Магазин выполняет регулярную проверку наличия обновлений и уведомляет вас, если доступна новая версия приложения и/или установленных элементов. 
Отключить это поведение можно в разделе [Уведомления] на странице [Параметры] главного окна приложения.

### Могу ли я добавить свой элемент в Магазин?

Конечно! Магазин открыт для всех желающих. Чтобы узнать подробности обратитесь к разработчику приложения через официальный телеграм-канал.
