# "This item is currently unsupported"

You typically receive this error when trying to download an unsupported item from the Store.

### Possible fixes

- Update the app to the latest version.
- Change the Store version control on the [Store/Settings] page. This will force the app to download the database from another source, which may contain a supported version of this item.

### If all else fails

- This item may not currently be supported by the app and will be supported in the future.

- An error may have occurred. Please report it to the app's technical support team, attaching the necessary logs and describing the problem in detail (this is important!). You can find the technical support address on the [About] page of the app's main window.
